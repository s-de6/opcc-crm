## Deploy steps (PowerShell as Administrator)

```

cd C:\Users\LENOVO\Downloads\opcc-crm-main\api
npm install
npx wrangler login
npx wrangler d1 execute opcc-crm-db --remote --file=src/db/migration-invoices.sql
npx wrangler deploy

cd ..\frontend
npm install
npm run build
npx wrangler pages deploy dist --project-name=opcc-crm --branch=production --commit-dirty=true
```
